Pour tester votre installation de Django, vous pouvez utiliser la commande `django-admin`qui sert entre autres choses à [[Créer un nouveau projet Django]].

Si vous tapez
```shell
django-admin --version
```

Vous devriez obtenir le numéro de version de Django:
```
6.0.4
```

# Choses à vérifier en cas d'erreur

- Est-ce que votre [[environnement virtuel#Activer un environnement virtuel|environnement virtuel est activé]] ?
- Est-ce que vous avez [[Installation de Django#Étape 2 Installer à l'aide de `pip`|installé Django dans cet environnement]] ?
