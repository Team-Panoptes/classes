Django est ce qu'on appelle une "framework web". C'est à dire un logiciel et du code vous permettant de créer des sites web à l'aide de Python.

Django fournit tout les outils de bases pour vous laisser vous concentrer sur les parties propres à votre application spécifique.

Sa philosophie est d'être "piles incluses", c'est-à-dire de vous proposer tout un ensemble d'outils déjà intégrés:

- Gestion des bases de données
- Utilisation de modèles HTML (templates)
- Interface d'administration
- Pages pré-faites pour les opérations les plus courantes
- ...

Ce n'est pas la seule option de framework en Python, mais c'est un projet très populaires avec une énorme communauté très active.
