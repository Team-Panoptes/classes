Dans Django, les modèles représentent les données qui viennent de la base de données (DB).

Ces données sont stockées dans différents types de base de données telles que MySQL, MariaDB, Oracle, PostgreSQL, SQLite, ou autre, mais on interagit avec au travers d'une couche spéciale, l'[[ORM]] ("Object Relational Mapping") qui traduit les tables de la DB sous forme d'objets Python.

On les retrouves souvent dans un fichier nommé `models.py`.
